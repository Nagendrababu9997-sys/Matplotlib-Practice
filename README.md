# Matplotlib-Practice

# 📊 Matplotlib Practice Repository

A collection of hands-on practice programs covering **Matplotlib**, the fundamental data visualization library in Python.

This repository demonstrates various plotting techniques, customization options, and visualization concepts used in **Data Analysis**, **Machine Learning**, and **Data Science**.

## 🚀 Skills Covered

- Line Charts
- Scatter Plots
- Bar Charts
- Histograms
- Pie Charts
- Box Plots
- Violin Plots
- Subplots
- Figure Styling
- Legends & Labels
- Grid & Axis Customization
- Color Maps
- Plot Customization

## 🛠 Tech Stack

- Python
- Matplotlib
- NumPy
- Pandas

## 🎯 Objective

To strengthen my data visualization skills through practical coding examples and build a solid foundation for Exploratory Data Analysis (EDA) and Machine Learning.
